Tips for JAWS Users

Here are some helpful hints to use this search form. The state select box is ordered with option groups so the New England states are grouped first in alphabetical order. Then the rest of the U.S. States are grouped in alphabetical order. Then Canadian providences are grouped in alphabetical order. So if you want to search for jobs only in Massachusetts it's the third option not twenith option.

Also to check the checkboxes you do not need to turn Forms Mode on. JAWS version 7.0 and up can check checkboxes without being in forms Mode. We are aware JAWS keeps repeating "Extended Select List Box" when it gets to the State Select box, currently the best thing to is to go into forms more to select the state or states you want to search in. To select multiple states once in Forms Mode press shift + F8 to go into Multiple select mode press space bar to select or unselect a state.
